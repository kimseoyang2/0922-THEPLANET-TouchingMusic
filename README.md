
# Touching Music – The Little Prince # 
신기한 VR 공간 속, 다양한 물건을 만지고 두드리며 상상력을 자극하자!


#### About Project
어린이들을 위한 가벼운 VR 음악 동화
#### Interaction
배경음악과 함께 다양한 소리를 가진 물체를 터치, 두들기기
#### Story
어린왕자는 어떻게 자신의 별로 돌아갔을까?
